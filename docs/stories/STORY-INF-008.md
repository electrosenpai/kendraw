## User Story

En tant que **dev Kendraw**,
je veux une **structure docs/ stable avec un template ADR prêt à l'emploi**,
afin de **acter les décisions architecturales au fil de l'eau dans un format standard**.

## Acceptance Criteria

- [ ] `docs/adr/` existe avec un template `0000-template.md`.
- [ ] `docs/retrospectives/` existe (vide, pour les rétros futures).
- [ ] Template ADR suit le format standard : Title, Status, Context, Decision, Consequences.
